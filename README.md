# Trackio – Smart Personal Finance & Expense Tracker

Trackio is a simple and powerful mobile application that helps users manage daily expenses, track income, savings, lending/borrowing, and get a clear overview of their financial life.

---

## Features

- **CRUD Transactions**  
  Add, edit, and delete:
  - Expenses  
  - Income  
  - Lent  
  - Borrow  

- **Lent & Borrow Completion**  
  Mark lent/borrowed transactions as completed when:
  - Someone pays you back  
  - Or you repay your debt  

- **Daily Expense Tracking**  
  Track how much you spend every day.

- **Category-wise Monthly Pie Chart**  
  Visualize monthly expenses by category.

- **Smart Dashboard**
  Quick overview of:
  - Remaining balance  
  - Today’s income  
  - Daily spending limit (for remaining days of the month)  
  - Total savings  
  - Total expense for the current month  

- **Savings Tracking**
  - Add previous savings (even without transaction history)  
  - Create, update, and delete new savings  

- **Monthly Overview**
  - Income  
  - Expense  
  - Remaining balance  

- **Guest Mode (No Data Loss)**
  - Start using the app as a guest  
  - Convert to a permanent account anytime without losing data  

- **Authentication**
  - Email & password  
  - Google Sign-In  
  - Secured with Firebase Authentication  

- **Cloud Database**
  - Powered by Firebase Firestore for secure data storage  

- **Daily Notifications**
  - Motivational reminders  
  - Encourage saving  
  - Remind users to add transactions  

---

## Tech Stack

- **Flutter** – Cross-platform mobile framework  
- **GetX** – State management & routing  
- **Firebase Authentication** – Login & user management  
- **Cloud Firestore** – Real-time database  
- **Local Notifications** – Daily reminders
